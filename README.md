Criei essa aplicação para reforçar os meus estudos em NodeJS, Express e SQLITE. 
O usuário conseguirá criar, editar, apagar ou marcar como concluida uma tarefa, com a persistência de dados SQLITE. 
As tarefas terão algumas validações de duplicata de nome, formato de data e valores. 
Estruturei o meu algoritmo seguindo um padrão UML, onde cada item tem a sua responsabilidade.
